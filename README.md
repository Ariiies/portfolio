## ğŸš€ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ariiies/mi-portafolio.git
   cd mi-portafolio
   ```

2. **Navigate to frontend directory**
   ```bash
   cd frontend
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start development server**
   ```bash
   npm run dev
   ```

5. **Open browser**
   ```
   http://localhost:5173
   ```
